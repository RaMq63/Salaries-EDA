# Salaries-EDA
This project analyzes a salary dataset to understand how different factors such as experience, job title, company size, and remote ratio actually affect salary levels. The analysis shows that company location is the strongest factor influencing salary, even more than experience or employment type.


### Dataset Features

The dataset had the following columns:

-work_year – Year the salary was paid
-experience_level – Junior / Mid / Senior / Expert
-employment_type – FT / PT / Contract / Internship
-job_title – Job position
-salary – Salary in original currency
-salary_currency – Original currency
-salary_in_usd – Converted salary in USD
-employee_residence – Country where the employee lives
-remote_ratio – 0%, 50%, 100% remote
-company_location – Where the company is based
-company_size – S / M / L

I Found: 
- Company Location is the strongest factor affecting salary,
Location effect is stronger than:

-Experience level
-Job title
-Remote ratio
-Company size
